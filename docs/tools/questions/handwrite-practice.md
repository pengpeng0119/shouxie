# 在线手写题练习

手写题不只是“能写出来”，更重要的是能快速验证边界情况和实现完整度。这个页面把题目描述、代码编辑和测试运行合到了一起，适合你在面试前做高频题的密集训练。

::: tip 使用建议
- 先自己完成代码，再点击“运行测试”
- 测试未通过时，优先看失败断言而不是直接看答案
- 练习完成后可以切换同分类的下一题继续刷题
:::
